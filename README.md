## Key Features

### Multi-Model Support
Connect to multiple AI platforms simultaneously using official APIs:
- OpenAI GPT (GPT-4, GPT-3.5)
- Anthropic Claude
- Google Gemini
- Groq
- Ollama (local models)
- Custom API endpoints and models

### Advanced Configuration
Fine-tune your AI interactions with:
- Temperature control
- Top-p (nucleus sampling) adjustment
- Custom system prompts
- Model-specific parameters

### Privacy First
- All chat history stored locally on your device
- No data collection or tracking
- Direct API communication only during active chats

### Modern Android Design
- Material You design language with dynamic theming
- Seamless dark mode support
- Adaptive UI without activity restarts
- Per-app language settings (Android 13+)

### Technical Excellence
Built with industry best practices:
- 100% Kotlin
- Jetpack Compose UI
- Single Activity architecture
- Modern App Architecture patterns
